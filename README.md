Регистрация
1. test_successful_registration_with_valid_data
Проверяет успешную регистрацию пользователя с корректными данными. После отправки формы пользователь создаётся и может войти в систему.
2. test_registration_fails_with_invalid_password
Проверяет, что регистрация невозможна при вводе некорректного пароля (например, слишком короткого). Отображается сообщение об ошибке.
Вход
3. test_login_via_main_page_login_button
Проверяет вход в систему через кнопку “Войти в аккаунт” на главной странице
4. test_login_via_personal_account_button
Проверяет вход через кнопку “Личный кабинет” с главной страницы.
5. test_login_via_registration_form_link
Проверяет возможность входа через ссылку “Войти” на странице регистрации.
6. test_login_via_password_recovery_form
Проверяет вход через форму восстановления пароля.
Личный кабинет
7. test_navigate_to_personal_account
Проверяет переход пользователя в раздел “Личный кабинет” после авторизации.
Переходы
8. test_navigate_from_profile_to_constructor_via_constructor_button
Проверяет переход из личного кабинета в конструктор через кнопку “Конструктор”.
9. test_navigate_from_profile_to_constructor_via_logo
Проверяет переход в конструктор через клик по логотипу сайта.
Выход
10. test_logout_from_personal_account
Проверяет успешный выход пользователя из системы через кнопку “Выход”. После выхода отображается кнопка входа.
Конструктор
11. test_switch_to_buns_section
Проверяет переключение на раздел “Булки” в конструкторе.
12. test_switch_to_sauces_section
Проверяет переключение на раздел “Соусы” в конструкторе.
13. test_switch_to_fillings_section
Проверяет переключение на раздел “Начинки” в конструкторе.
